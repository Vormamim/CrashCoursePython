# Student Worksheet: Lesson 7 Loop Launch (`for`)

## YA Theme Context
This lesson is inspired by maze-run YA missions where checkpoint systems monitor progress.
You are coding the scanner that checks each checkpoint in order.
Your loop represents repeated patrol checks during the run.

## Theory
- A loop repeats code.
- A `for` loop repeats over a sequence.
- `range(start, stop)` includes start and stops before stop.

## Scenario Brief (Explicit)
You are running a maze checkpoint scanner.

Required outcome:
- Print checkpoints 1 to 8
- Print "Hazard scan required" on every 3rd checkpoint

## Good vs Weak Gemini Prompts
Weak:
- "Write my assignment"
Good:
- "Show only the `for` loop step for checkpoints 1-8 and explain `% 3 == 0` in beginner words."

## Deliverable (Measurable)
1. Uses one `for` loop
2. Uses `range(1, 9)` or equivalent
3. Uses a condition inside the loop
4. Includes one improved prompt example
5. Includes one test output screenshot

## Scaffolded Task
1. Print title line.
2. Write `for` loop for checkpoints.
3. Print checkpoint value in loop.
4. Add conditional for every 3rd checkpoint.
5. Test and record output.

## Reflection
- What did your loop variable represent?
- How did you improve a weak prompt?
