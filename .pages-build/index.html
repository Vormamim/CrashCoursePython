What this course is about

Each lesson introduces a concept, shows why it matters in the story, and ends with something you can run and explain.

Story-driven learning

Every task is framed as a safe mission: arenas, terminals, factions, scoreboards, portals, and looping simulations.

Visible progression

Learn how to move from variables and output to branching, data structures, and loops in a clear, ordered pathway.

Independent practice

Each lesson includes a mission goal and starter code!

Responsible AI use

Later lessons teach students how to use Gemini for hints, reasoning, and debugging without outsourcing the whole solution.
