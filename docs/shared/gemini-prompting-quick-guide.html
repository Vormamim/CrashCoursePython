Quick reference

Gemini Prompt Cheat Sheet

Use Gemini for hints, planning, and debugging. Do not ask for full assignment answers.

Understand the task first

Use when the question is unclear and you need a plain-language breakdown.

Prompt
I am a beginner Python student.
Explain this task in simple words.
Then list:
1) what inputs I have,
2) what output I should produce,
3) the steps I should try.
Do not give full code yet.

Plan before coding

Use when you want a clear strategy before typing code.

Prompt
Help me make a Python plan for this problem:
[paste problem]

Use beginner concepts only from this course:
- variables
- input + conversion
- if / elif / else
- lists / tuples
- loops (if lesson 7+)

Give pseudocode, not full code.

Debug an error

Use when your code crashes or output is wrong.

Prompt
I need debugging help.
Here is my code:
[paste code]

Here is the error/output:
[paste error]

Please:
1) explain the bug simply,
2) point to the exact line,
3) suggest the smallest fix,
4) give one test case to confirm.

Explain code line by line

Use when you want to understand code deeply.

Prompt
Explain this Python code line by line for a beginner.
For each line, tell me:
- what it does
- why it matters
- one common mistake

Code:
[paste code]

Improve print output

Use when your console output looks messy or unclear.

Prompt
Help me improve the print output in this Python code.
Keep the same logic, only improve readability.
Use beginner-friendly print statements.

Code:
[paste code]

Hints only, no full answer

Use when you want support while still doing your own work.

Prompt
Do NOT give me the final solution.
Give me 3 hints from easy to harder.
After the hints, ask me one checkpoint question
to test if I understand before I continue coding.
