Quick reference

Python Student Cheat Sheet

Very short theory + sample code. Use copy on any snippet and paste into the IDE.

Variables

Store data in named values so you can reuse and update it.

Sample
name = "Ari"
score = 12
score = score + 3
print(name, score)

Input + conversion

Use input() to collect text, then convert to numbers when needed.

Sample
age_text = input("Age: ")
age = int(age_text)
print(age + 1)

If / elif / else

Branch your logic so code reacts to conditions.

Sample
power = 72
if power >= 80:
    print("Elite")
elif power >= 60:
    print("Ready")
else:
    print("Train")

Lists + index

Lists hold ordered items. Use indexes to read or update positions.

Sample
teams = ["Falcon", "Wolf", "Stag"]
teams[1] = "Raven"
print(teams[0])
print(len(teams))

For loop

Repeat a known number of times.

Sample
for checkpoint in range(1, 6):
    print("Checkpoint", checkpoint)

While loop

Repeat while a condition is true. Always update the loop state.

Sample
countdown = 3
while countdown > 0:
    print(countdown)
    countdown -= 1
print("Launch")

Validation pattern

Keep asking until input is valid.

Sample
score = int(input("Score 0-100: "))
while score < 0 or score > 100:
    score = int(input("Try again: "))
print("Accepted", score)

Debug quick check

Print key values to trace logic quickly.

Sample
power = 55
risk = 100 - power
print("power=", power)
print("risk=", risk)
