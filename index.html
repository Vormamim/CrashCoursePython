What this course is about

Gemini Prompt Cheat Sheet (Beginner Python)

Use these prompt templates in later lessons. Ask for help with thinking, debugging, and explaining — not for full solutions.

1) Understand the task first

Use when: You are not sure what the problem is asking.

I am a beginner Python student.
Explain this task in simple words.
Then list:
1) what inputs I have,
2) what output I should produce,
3) the steps I should try.
Do not give full code yet.

2) Plan before coding

Use when: You want a clear step-by-step approach.

Help me make a Python plan for this problem:
[paste problem]

Use beginner concepts only from this course:
- variables
- input + conversion
- if / elif / else
- lists / tuples
- loops (if lesson 7+)

Give pseudocode, not full code.

3) Debug an error

Use when: Your code crashes or gives the wrong output.

I need debugging help.
Here is my code:
[paste code]

Here is the error/output:
[paste error]

Please:
1) explain the bug simply,
2) point to the exact line,
3) suggest the smallest fix,
4) give one test case to confirm.

4) Explain code line by line

Use when: You copied or wrote code and want to understand it.

Explain this Python code line by line for a beginner.
For each line, tell me:
- what it does
- why it matters
- one common mistake

Code:
[paste code]

5) Improve output formatting

Use when: Your print output is messy or unclear.

Help me improve the print output in this Python code.
Keep the same logic, only improve readability.
Use beginner-friendly print statements.

Code:
[paste code]

6) Ask for hints, not answers

Use when: You want to learn without outsourcing the solution.

Do NOT give me the final solution.
Give me 3 hints from easy to harder.
After the hints, ask me one checkpoint question
to test if I understand before I continue coding.
