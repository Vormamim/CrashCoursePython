# Teacher Key: Lesson 8

## Expected Technical Outcomes
- Uses `while countdown >= 1` (or equivalent)
- Decrements counter in loop
- Prints final post-loop line

## Objective Answers
1. Infinite loops happen when condition never becomes false
2. Counter update is usually `countdown = countdown - 1`
3. Final line runs after loop exits

## Common Errors
- Forgetting counter update
- Wrong comparison operator
- Printing final line inside loop by mistake
