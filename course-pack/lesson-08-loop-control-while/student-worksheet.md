# Student Worksheet: Lesson 8 Loop Control (`while`)

## Instructions
Follow the worksheet steps in order and complete each task using the required variables, inputs, and outputs.

## Theory
- `while` repeats while a condition is true.
- A control variable must change to avoid infinite loops.

## Scenario Brief (Explicit)
You run an arena launch countdown.

Required outcome:
- Print 10 down to 1
- Print `Launch` when countdown ends

## Good vs Weak Gemini Prompts
Weak:
- "Fix this"
Good:
- "My while loop does not stop. Explain the exact line causing it and one beginner-safe fix."

## Deliverable (Measurable)
1. Uses one `while` loop
2. Uses a countdown variable
3. Updates variable each cycle
4. Prints `Launch` after loop
5. Includes one improved Gemini debugging prompt

## Scaffolded Task
1. Set countdown variable.
2. Write while condition.
3. Print countdown inside loop.
4. Decrease countdown inside loop.
5. Add final `Launch` line.

## Reflection
- Why did your loop stop at the correct point?
- How did a better prompt help your debugging?
