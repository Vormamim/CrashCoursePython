# Student Worksheet: Lesson 1

## Theory (What You Need to Know)
- A variable is a named box that stores data.
- Python data types used today: `int`, `float`, `str`, `bool`.
- Reassignment updates a variable when something changes.

## Metalanguage (Key Terms + Examples)
- `variable`: A named value your code can use later.
Example: `score = 10`
- `assignment`: Putting a value into a variable using `=`.
Example: `district = 7`
- `reassignment`: Changing a variable after it already has a value.
Example: `score = score + 2`
- `data type`: The kind of value stored.
Example: `7` is `int`, `8.5` is `float`, `"Raven"` is `str`, `True` is `bool`.
- `statement`: A complete instruction line Python executes.
Example: `print("Mission Start")`
- `output`: Information shown to the user.
Example: `print("Score:", score)`

## Relatable Examples
- Scoreboard update: `score = 8.5` then `score = score + 1.5`.
- Team name text: `team_name = "Raven"`.
- Mission status: `is_active = True`.
