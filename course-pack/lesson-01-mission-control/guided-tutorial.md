# Guided Tutorial: Lesson 1 Mission Control

Use this before the worksheet. Students type one step at a time.

## Goal
Build a simple arena profile program using variables, types, and output.

## Step-by-step Build
1. Add this line:
```python
print("Arena Mission Console")
```
Explainer: `print()` shows text to the screen.

2. Add this line:
```python
participant_name = "Ari"
```
Explainer: This creates a `str` variable (text).

3. Add this line:
```python
district = 7
```
Explainer: This creates an `int` variable (whole number).

4. Add this line:
```python
score = 8.5
```
Explainer: This creates a `float` variable (decimal number).

5. Add this line:
```python
is_active = True
```
Explainer: This creates a `bool` variable (`True` or `False`).

6. Add this line:
```python
score = score + 1.5
```
Explainer: This is reassignment. It updates `score` using the old value.

7. Add this line:
```python
print("Name:", participant_name)
```
Explainer: This prints a label and a variable value.

8. Add this line:
```python
print("District:", district)
```
Explainer: You can print numbers and text in one print statement.

9. Add this line:
```python
print("Score:", score)
```
Explainer: Updated values print after reassignment.

10. Add this line:
```python
print("Active:", is_active)
```
Explainer: Booleans print as `True` or `False`.

## Quick Check
- Did you create 4 variables?
- Did you update `score` once?
- Do you see 4 labeled output lines?
