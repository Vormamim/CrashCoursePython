# Lesson 7: Loop Launch (`for`)

## Duration
- 60 minutes total
- 40 minutes coding

## Learning Goals
- Understand repetition in code
- Use `for` loops with `range()`
- Use loop variables to control output
- Prompt Gemini for one loop step without requesting full solutions

## Scenario Brief (Explicit)
Theme: Maze Runner checkpoint scanner.
Students build a terminal that prints checkpoint IDs and basic status lines.

Required elements:
- `for` loop over `range()`
- At least one loop variable in output
- One condition inside a loop

## End-of-Lesson Deliverable (Measurable)
1. Program prints checkpoints 1-8
2. Every third checkpoint prints "Hazard scan"
3. Student submits two prompts (weak vs improved)
4. Student explains what loop variable means

## Good vs Weak Gemini Prompts
Weak:
- "Write my whole checkpoint program"
Good:
- "Teach me how to print checkpoints 1-8 with a `for` loop and mark every third checkpoint as hazard. Give only this step, with explanation and expected output."

## Schedule
- 0-10 min: Loop concept and prompt quality demo
- 10-20 min: Guided tutorial
- 20-55 min: Worksheet coding
- 55-60 min: Exit check
