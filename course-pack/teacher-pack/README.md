# Python Crash Course (6 Lessons, Age 13+)

This pack delivers six 60-minute lessons with 40 minutes of coding per lesson.

## Constraints Used
- No loops or iteration
- No user-defined functions
- No dictionaries
- No file handling

## Audience
- 13 year olds with above-average maths, logical thinking, and English

## Theme
- Young adult movie inspired coding missions (classroom-safe references)

## Lesson Timing Blueprint (Every Lesson)
- 0-8 min: Hook and mini-demo
- 8-18 min: Direct instruction
- 18-58 min: Coding mission (40 min)
- 58-60 min: Exit check
