# Lesson 8: Loop Control (`while`)

## Duration
- 60 minutes total
- 40 minutes coding

## Learning Goals
- Use `while` loops safely
- Build/update a counter variable
- Avoid infinite loops
- Use Gemini to debug loop conditions without outsourcing full work

## Scenario Brief (Explicit)
Theme: Arena countdown lock.
Students build a countdown terminal from 10 to 1 and print "Launch".

Required elements:
- `while` loop with clear stop condition
- Counter update each cycle
- Final line after loop ends

## End-of-Lesson Deliverable (Measurable)
1. Countdown prints 10 to 1
2. Program prints "Launch" once
3. Student explains why loop stops
4. Student provides one weak and one improved debugging prompt

## Good vs Weak Gemini Prompts
Weak:
- "My loop is broken. Fix it."
Good:
- "My while loop never stops. Here is my code and output. Explain which line causes infinite looping and show one beginner fix."

## Schedule
- 0-10 min: while loop mechanics
- 10-20 min: guided tutorial
- 20-55 min: worksheet coding
- 55-60 min: exit check
