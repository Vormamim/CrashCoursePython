# Teacher Key: Lesson 9

## Expected Technical Outcomes
- `for` loop used for fixed repeats
- `while` loop used for validation retries
- Correct eligibility counter update

## Objective Answers
1. Use `for` when repeat count is known
2. Use `while` when stop condition depends on state
3. Validation loop exits once score is in allowed range

## Common Errors
- Forgetting to re-read input inside validation loop
- Counter increment in wrong branch
- Mixed indentation in nested blocks
